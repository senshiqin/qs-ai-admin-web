# QS-AI Admin Web

Vue 3 + TypeScript + Element Plus frontend for the QS-AI Spring Boot 3.2.5 backend.

## Run

```powershell
cd qs-ai-admin-web
npm install
npm run dev
```

The Vite dev server proxies `/api` and `/health` to `http://localhost:8080`.

## Pages

- Login
- Dashboard
- AI Chat
- RAG Retrieval / Streaming QA
- Knowledge Documents
- Model Config
